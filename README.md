# 🇫🇷 Französisch Lernplattform - Vollversion

Eine professionelle, interaktive Lernplattform für französische Verben und Grammatik mit 4 verschiedenen Lernmodi.

## 🚀 Neue Features (Optimierte Version)

### ✨ **Hauptseite mit Navigation**
- Übersichtliche Startseite mit allen Lernmodulen
- Fortschritts-Tracking über alle Module
- Responsive Design für alle Geräte
- Lerntipps und optimale Lernreihenfolge

### 📚 **Erweiterte Lernkarten (32 Verben)**
- **32 wichtige französische Verben** (statt 22)
- **Touch-Gesten**: Wischen für Navigation
- **Schwierigkeitsstufen**: Alle, Unregelmäßige, Regelmäßige, Modalverben
- **Autoplay-Modus** für automatisches Durchgehen
- **Fortschritt speichern** mit LocalStorage
- **Achievements** für Motivation
- **Tastatur-Navigation** (Pfeiltasten, Leertaste)

### 🧠 **Interaktives Quiz**
- **3 Schwierigkeitsstufen**: 10/15/20 Fragen
- **4 Fragetypen**: Konjugation, Übersetzung, Grammatik, Gemischt
- **Zeitlimit optional**: Kein Limit, 30s oder 60s pro Frage
- **Sofortiges Feedback** mit detaillierten Erklärungen
- **Detaillierte Auswertung** mit Bewertung
- **Fortschrittsbalken** und Punktesystem

### 🎮 **Erweitertes Memory-Spiel**
- **3 Schwierigkeitsstufen**: 8/12/15 Verb-Paare
- **Statistiken**: Beste Zeit, Durchschnitt, Gesamtspielzeit
- **Achievement-System** für Erfolge
- **Pause-Funktion** und **Tipp-System**
- **Genauigkeits-Anzeige** und Bewertungen
- **Mobile Touch-Optimierung**

### 📖 **Grammatikregeln (Unverändert)**
- 15 wichtige Regeln und Ausnahmen
- Farbkodierte Kategorien
- Lerntipps und Eselsbrücken

## 🚀 Dauerhafte Bereitstellung mit GitHub Pages

### Schritt 1: GitHub-Account erstellen
1. Gehe zu [github.com](https://github.com)
2. Klicke auf "Sign up" (falls du noch keinen Account hast)
3. Erstelle einen kostenlosen Account

### Schritt 2: Neues Repository erstellen
1. Klicke auf das **+** Symbol oben rechts
2. Wähle **"New repository"**
3. Repository-Name: `franzoesisch-lernplattform` (oder einen anderen Namen)
4. Setze Häkchen bei **"Add a README file"**
5. Klicke **"Create repository"**

### Schritt 3: Dateien hochladen
1. **ZIP entpacken** und alle HTML-Dateien extrahieren
2. Klicke auf **"uploading an existing file"** oder **"Add file" → "Upload files"**
3. Ziehe alle Dateien in das Fenster:
   - `index.html` (Hauptseite)
   - `cards.html` (Erweiterte Lernkarten)
   - `quiz.html` (Interaktives Quiz)
   - `memory.html` (Erweitertes Memory)
   - `regeln.html` (Grammatikregeln)
   - `README.md` (Diese Anleitung)
4. Schreibe eine Commit-Nachricht: "Französisch Lernplattform hinzugefügt"
5. Klicke **"Commit changes"**

### Schritt 4: GitHub Pages aktivieren
1. Gehe zu **Settings** (oben im Repository)
2. Scrolle runter zu **"Pages"** (links im Menü)
3. Bei **"Source"** wähle **"Deploy from a branch"**
4. Bei **"Branch"** wähle **"main"**
5. Klicke **"Save"**

### Schritt 5: Deine Lernplattform ist online! 🎉
Nach 1-2 Minuten ist deine Website erreichbar unter:
```
https://dein-github-name.github.io/franzoesisch-lernplattform/
```
